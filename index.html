SDK

SA SCREEN DETECTOR SDK v1.0 для Android поставляется в виде дистрибутива проекта библиотеки Android. Он написан на Java и C и доступен для Android 5.1 и более поздних версий. Библиотека позволяет добавить в любое андроид-приложение логику по захвату отраженного (в зеркале) изображения экрана с последующим анализом экрана на дефекты. Первичный захват и первичный анализ происходит непосредственно на смартфоне, окончательный анализ и поиск результата происходят на сервере.
